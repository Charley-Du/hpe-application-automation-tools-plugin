This step is responsible on fetching of pull requests from SCM tool and injection to ALM Octane.
Every time pull requests are sent to ALM Octane, last update time is saved, and next time , only pull requests and commits that were updated after it, will be sent to ALM Octane.

Supported version of ALM Octane : 15.0.49

You can change behaviour of the step by adding the following parameters to the job:
  • pullrequests_min_update_time : use this parameter to override last update time that is updated after each successful injection to ALM Octane.

  • pullrequests_page_size : page size is defined on each request to SCM tool, so a tool return this count of items (pullrequest or commits). Default is 30.

  • pullrequests_max_pr_to_fetch: Max number of pull-request to fetch. Default is 100.

  • pullrequests_max_commits_to_fetch : Max number of commits for each pull-request to fetch. Default is 100.
